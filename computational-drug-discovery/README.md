# Computational Drug Discovery (In Progress)

This project explores how computational tools can assist early-stage drug discovery by analyzing molecular structure data and predicting potential interactions between molecules and biological targets.

The goal of this project is to build a lightweight Python-based workflow that demonstrates how algorithms and data analysis can help screen chemical compounds before laboratory testing.

## Objectives
- Analyze molecular datasets using Python
- Explore molecular properties that influence drug binding potential
- Visualize chemical structure data
- Build a simple prediction model for molecular activity

## Tools & Technologies
- Python
- RDKit (molecular analysis)
- Pandas / NumPy
- Matplotlib for visualization
- Jupyter Notebook

## Project Status
Currently in development. Initial steps include dataset exploration and building a workflow for analyzing molecular descriptors.

## Future Development
- Implement machine learning models to predict molecular activity
- Visualize molecular similarity networks
- Explore potential drug candidates from public chemical datasets

## Motivation
This project reflects my interest in the intersection of chemistry, computer science, and medicine. Computational methods are becoming increasingly important in pharmaceutical research, allowing scientists to analyze thousands of molecular candidates before conducting laboratory experiments.

By combining programming and chemical analysis, I aim to better understand how computational tools can accelerate biomedical discovery.
